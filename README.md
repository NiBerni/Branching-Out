# Branching-Out
Project during the MSIT program for Software Engineering. Getting started with branching

A simple project to learn branching.
The main purpose was the use of branches to implement new features.
The project is a simple filter script that can read user data from a json
and show a filtered output of the data in the terminal.
It started with one filter option, the name.
Than the project was expanded to include more filters.
First an option for filtering by age was added via a new feature branch.
The merged was done in PyCharm.
The second option was to add an option for filtering by E-mail.
This time, the merge was done on GitHub via a pull request.

The final refactoring of my own code was done in the main branch.
The provided code was left unchanged and not refactored.
